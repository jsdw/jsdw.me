Content for [jsdw.me](https://jsdw.me). Please submit a PR if you'd like to correct spelling mistakes and such.

# Building

```
# Install gutenberg (following for Ubuntu 18.04):
wget https://github.com/getzola/zola/releases/download/v0.3.3/gutenberg-v0.3.3-x86_64-unknown-linux-gnu.tar.gz
tar -xf gutenberg-v0.3.3-x86_64-unknown-linux-gnu.tar.gz

gutenberg serve # locally host page
gutenberg build # build it
```
